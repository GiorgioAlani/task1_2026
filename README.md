#  Оценка площади круга методом Монте-Карло

Программа для вычисления площади круга с использованием статистического метода Монте-Карло.

##  Описание

Программа генерирует случайные точки в прямоугольной области и подсчитывает, какая доля из них попадает внутрь круга. На основе этой доли вычисляется приблизительная площадь круга.
